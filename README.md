# DaMealZone app 👋

The project is a simple macro tracking app called DaMealZone that allows users to add meals and track their macros (calories, protein, carbs, fat). The app has a home screen that displays the current date, a grid of macro cards showing the total macros for the day, and a list of recent meals. There is also a screen for adding new meals and a screen for viewing all meals.
